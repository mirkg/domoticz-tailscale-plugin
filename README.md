# Tailscale Domoticz Plugin

## Overview

Thit Domoticz plugin allow to communicate from Domoticz to Tailscale and monitor status of connection and devices.

## Run

1. Clone repository into your domoticz plugins folder
```
cd domoticz/plugins
git clone https://github.com/mirkg/domoticz-tailscale-plugin.git tailscale
```
2. Restart domoticz
3. Make sure that "Accept new Hardware Devices" is enabled in Domoticz settings
4. Go to "Hardware" page and add new item with type "Tailscale"
